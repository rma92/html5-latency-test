Targets JSON

Provide an array of { "region": "…", "host": "…" }. You can override this textarea by passing a ?list= URL parameter containing URL‑encoded JSON.

Tip: we send a small GET to the root of each host over HTTPS with mode:"no-cors" and abort immediately after headers arrive. This approximates TCP+TLS handshake + first byte time without downloading bodies.
